Lesson Two: Your first words!

Let's learn your first 27 words/modifiers in Shmorsh!

All of these words each correspond to one letter of the alphabet.

a = Life (Noun), Live (Verb), Alive (Adj) Lively (Adv)

b = Time (Noun), Time (Verb), Early (Adj), Timely (Adv)

c = Food (Noun), Eat (Verb), Hungry (Adj), Tastefully (Adv)

d = Water (Noun), Drink (Verb), Thirsty (Adj), Thirstily (Adv)

e = Air (Noun), Breathe (Verb), Fresh (Adj), Freely (Adv)

f = Fire (Noun), Burn (Verb), Hot (Adj), Hotly (Adv)

g = Ice (Noun), Freeze (Verb), Icy (Adj), Icily (Adv)

h = Earth (Noun), None Specific (Verb), None Specific (Adj), None Specific (Adv)

i = Stone (Noun), None Specific (Verb), None Specific (Adj), None Specific (Adv)

j = Grass (Noun), None Specific (Verb), None Specific (Adj), None Specific (Adv)

k = Legs (Noun), Walk (Verb), Strong (Adj), Swiftly (Adv)

l = Speed (Noun), Move (Verb), Fast (Adj), Rapidly (Adv)

m = Like (Noun), Like (Verb), Liked (Adj), None Specific (Adv)

n = Plant (Noun), Grow (Verb), Healthy (Adj), Naturally (Adv)

o = Family (Noun), Love (Verb), Close (Adj), Deeply (Adv)

p = Mom (Noun), None Specific (Verb), None Specific (Adj), None Specific (Adv)

q = Dad (Noun), None Specific (Verb), None Specific (Adj), None Specific (Adv)

r = Daughter (Noun), None Specific (Verb), None Specific (Adj), None Specific (Adv)

s = Son (Noun), None Specific (Verb), None Specific (Adj), None Specific (Adv)

t = Light (Noun), Illuminate (Verb), Bright (Adj), Radiantly (Adv)

u = Creation (Noun), Make (Verb), Made (Adj), None Specific (Adv)

v = None Specific (Noun), Is (Verb), None Specific (Adj), None Specific (Adv)

w = None Specific (Noun), Want (Verb), None Specific (Adj), None Specific (Adv)

x = None Specific (Noun), Go (Verb), None Specific (Adj), None Specific (Adv)

y = Speech (Noun), Communicate (Verb), Verbal (Adj), Verbally (Adv)

z = Listening (Noun), Hear (Verb), Attentive (Adj), Attentively (Adv)

Add these words if there is no consonant after and/or they are vowels, for example age = life.

If they are consonants and the word after is a consonant, you add an 'e' after the letter, for example bege = time.

And word number 27 is our first MODIFIER (Prefixes/Suffixes).

an = Opposite of, goes BEFORE words. For example. tege = light, antege = dark.
